## Catalog Description

Software engineering principles and practices. Software development lifecycle. Requirements elicitation, design, implementation, testing, and deployment. Version control, container technologies, issue tracking, continuous integration, and testing. Project management and professional ethics. Collaborative software development, teamwork, and professional communication. Semester-long team-based software development project.
