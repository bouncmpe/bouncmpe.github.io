## Catalog Description

Bayes decision theory. Parametric and nonparametric methods. Linear discriminant functions. Higher order discriminants with emphasis on artificial neural network-based learning methods. Unsupervised learning and clustering. Case study: Vision.

## Course Learning Outcomes

- Understand Bayes decision theory for classification  
- Learn parametric and nonparametric pattern recognition methods  
- Understand linear discriminant functions  
- Learn higher-order discriminants and ANN-based approaches  
- Understand unsupervised learning and clustering  
- Explore case studies, especially in computer vision
