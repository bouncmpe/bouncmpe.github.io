## Catalog Description

Machine learning approaches using Bayesian statistics. Graphical models — directed and undirected models, learning and inference. Hidden Markov Models, Linear Dynamical Systems, Kalman filtering and smoothing, message passing algorithms. Junction Tree, factor graphs, sum-product, hierarchical Bayesian modeling, Expectation-Maximisation, and Variational Approximation techniques.

## Course Learning Outcomes

- Understand Bayesian approaches to machine learning  
- Learn graphical models (directed & undirected) and inference methods  
- Understand Hidden Markov Models and Linear Dynamical Systems  
- Learn Kalman filtering, smoothing, and message-passing algorithms  
- Understand Junction Tree, factor graphs, and sum-product algorithm  
- Learn hierarchical Bayesian modeling  
- Gain familiarity with EM and variational approximation techniques
